# Extensions catalog

The wrapper bundles **47 extensions**, registered as upstream pi `ExtensionFactory` functions and passed to `main()` via `bundledExtensionFactories` (`src/extensions/index.ts`). They are split into three groups in `definitions-group-{a,b,c}.ts` plus the `subagent` extension built separately.

This page lists every extension with a one-line purpose and its key source files, grouped by area. For how they're registered and how to add one, see [Architecture → Extension system](../architecture/overview.md#extension-system).

> Note: `src/extensions/interview/` contains a full extension (settings, server, form assets) and its defaults are wired into `default-settings.ts`, but it is **not** in any active A/B/C group, so it does not register at runtime. Treat it as bundled-but-dormant code.
